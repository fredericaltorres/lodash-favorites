# lodash-favorites
- My favorites lowdash functions 
- Tested with Mocha and [Chai](https://www.chaijs.com/)

# Setup 

```bash
npm install --save lodash
npm install --save-dev mocha  
npm install --save-dev chai
npm install --save-dev mockery
```
